# Institute of Digital Risk (IDR) – Homepage Design

## Overview

This project implements a responsive single-page homepage for the **Institute of Digital Risk (IDR)**, an industry-led training and deployment institute focused on digital, cyber, and AI risk.

The webpage introduces IDR’s mission, training model, community, and contact pathway through a clear and structured layout. The design focuses on clarity, accessibility, and responsiveness across desktop and mobile devices.

---

## Features

### 1. Hero Section

* Headline: **“Advancing the Future of Digital Risk”**
* Brief description of IDR’s mission and training approach
* Call-to-action button linking to program details

### 2. About IDR

Explains the role of IDR as an **industry-led training and deployment institute** preparing professionals for digital, cyber, and technology risk roles in regulated environments.

Highlights the connection between:

* Academic collaboration
* Real-world industry practice

### 3. Service Pillars / Model

Three key operational pillars are presented as cards:

**Academy**

* Bootcamps and training programs
* Digital risk and AI governance education

**Innovation & Incubation**

* Development of risk frameworks and digital resilience models
* Research and experimentation in emerging risk areas

**Advisory Services**

* Organisational support implementing frameworks such as:

  * NIST Cybersecurity Framework
  * ISO/IEC 27001
  * NIS2 Directive

### 4. Community

Describes the professionals served by IDR:

* Students
* Early-career professionals
* Practitioners in regulated industries such as financial services and critical infrastructure.

### 5. Contact / Call-to-Action

A simple form allows users to register interest by submitting:

* Name
* Email
* Message

---

## Technical Implementation

The project follows modern front-end development practices using **vanilla web technologies**.

### HTML

* Semantic HTML5 structure:

  * `header`
  * `nav`
  * `main`
  * `section`
  * `footer`

### CSS

* Responsive layout using **Flexbox and CSS Grid**
* Custom color variables for design consistency
* Hover states for navigation and buttons
* Mobile-friendly layout using media queries

### JavaScript

* Smooth scrolling for navigation links
* Lightweight client-side interaction

---

## Design System

### Color Palette

| Color      | Purpose                         |
| ---------- | ------------------------------- |
| Navy       | Primary background and branding |
| Orange     | Call-to-action highlights       |
| White      | Content backgrounds             |
| Grey tones | Supporting text and borders     |

### Typography

* **DM Sans** – primary body font
* **DM Serif Display** – headings and emphasis

### Logo

The IDR logo appears in the navigation bar and follows a **16:8 ratio** to maintain clarity across screen sizes.

## Responsive Design

The homepage adapts to different screen sizes using media queries:

Desktop:

* Full navigation menu
* Multi-column service cards

Mobile:

* Simplified layout
* Stacked content sections
* Scaled logo and typography

## Conclusion

This project demonstrates the design and development of a clean, responsive landing page for a professional institute. The implementation focuses on clear communication of IDR’s mission, structured information architecture, and accessible user interaction using only **HTML, CSS, and JavaScript**.
